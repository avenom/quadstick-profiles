## Как добавить сторонний профиль управления в QuadStick Manager Program (QMP)


1. Скопируйте ссылку стороннего профиля из Google Docs.

2. Подключите QuadStick к ПК по USB и откройте программу QMP.

3. В правом окне нажмите Add Game to User List, вставьте скопированную ссылку и нажмите OK.

4. Перенесите добавленный профиль из правого окна в левое окно In QuadStick.

5. Нажмите Save Preferences to QuadStick, дождитесь, когда индикаторы на QuadStick перестанут моргать, запомните номер профиля и закройте программу QMP.
